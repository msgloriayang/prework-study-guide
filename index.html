HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Canvas Style Sheets.
  • CSS is the style of the webpage.

GIT

  • There are many git codes that you can use.
  • I find that the easiest way to clone a repository is to take an https url or ssh code from git hub.
  • Git clone insert https url or ssh code.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript is how the user interacts with the webpage.
  • A great example of Javascript would be an animated image.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Once the class element is added into the html page, the css rule will apply to the assigned class selector.
